August

I've created it before...
The joy of a peaceful core
To perfection I let it rise
As it took with it my pride
I've nurtured it before...
As I carefully sat in awe
The impeccable feeling of whole
I've felt it in my soul
I've taught it before...
Understanding the beauty it stored
The lessons of growth
Wisdom. versus. Know
I've lost it before...
The last memory to recall
Pain is stalled
Like a never ending fall
Yes, I've created it before
The glitter that lacked gold
